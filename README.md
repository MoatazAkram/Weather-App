# Weather App
A responsive weather forecast web app that shows current weather and a 3-day forecast for any city around the world.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Features
- Auto-detects user location via Geolocation API
- Real-time city search with debouncing
- 3-day weather forecast
- Error handling for invalid cities or connection issues
- Auto-refresh every 30 minutes
- Loading state while fetching data
